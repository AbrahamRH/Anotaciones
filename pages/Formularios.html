Formularios

Para brindarle estilos a nuestros formularios, es necesario agregar un contenedor dentro de la etiqueta <form> con la clase form-group esto se hace para cada elementro que se desee colocar en el formulario, cada elemento con su propio contenedor, 치demas, cada elemento debe tener la clase form-control. En cada elemento (input) de un formulario, se agrega la clase form-control abarcar el 100% de el espacio.

Para los elementos checkbox y los radio buttons, el contenedor debe llamarse form-check, en lugar de form-group, y a los <label>, deben de tener la clase form-check-label, y a cada elemento de checkbox, se le agrega la clase form-check-input.
