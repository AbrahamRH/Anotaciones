Formularios

Para brindarle estilos a nuestros formularios, es necesario agregar un contenedor dentro de la etiqueta <form> con la clase form-group esto se hace para cada elemento que se desee colocar en el formulario, cada elemento con su propio contenedor, ádemas, cada elemento debe tener la clase form-control. En cada elemento (input) de un formulario, se agrega la clase form-control abarcar el 100% de el espacio.

Para los elementos checkbox y los radio buttons, el contenedor debe llamarse form-check, en lugar de form-group, y a los <label>, deben de tener la clase form-check-label, y a cada elemento de checkbox, se le agrega la clase form-check-input.

Formularios inline

Para crear un formulario de tipo inline, se le debe agregar la clase form-inline a la etiqueta <form>, destro de esta etiqueta se añaden los elementos normalmente sin un contenedor intermedio a excepción de los checkbox o radio buttons.

Formularios input groups

Los grupos de inputs, sirven para colocar labels en los formularios de tipo input, de una manera más estética, estos se crean encerrando los inputs en un contenedor con la clase input-group en la que estará el el elemento input con la clase form-control y junto con otro contenedor con la clase input-group-xxxxxx donde xxxxxx es prepend o append, para antes o despues, segun sea el caso. Finalizando, dentro de este contenedor debera ir una etiqueta <span> siendo este el label del input con la clase input-group-text.

Tambien se pueden poner las clases input-group-lg y input-group-sm para modificar los tamaños, estas clases deben agregarse en el contenedor principal de el input.

Ádemas se pueden poner botones en lugar de label que pueden usarse para barras de busqueda, unicamente agregando un boton con sus clases,veasé en Botones, en lugar de el span.
