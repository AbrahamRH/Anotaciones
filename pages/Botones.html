Componentes Botones

Bootstrap nos brinda la opción de crear botones gracias a la clase btn que debe estar dentro de la etiqueta <button>, tambien podemos tener distintos tipos de colores en los botones, veasé en colores.

Creando botones

Para crear un boton no unicamente se utiliza la etiqueta <button>, tambien se puede crear con la etiqueta <input> y con el tipo submit o button, al igual que el estilo de botones se puede dar en enlaces con la etiqueta <a>, y esto se da usando clases btn y btn-xxxxx, donde xxxxx es el color.

Botones outline

A diferencia de los otros botones, estos unicamente mantienen su borde, y se crean con la clase btn-outline-xxxxx, donde xxxxx es el color, y se ven de la siguiente manera.

Tamaño de botones

Boostrap nos permite crear botones con sus distintos tamaños, veasé en tamaños, con la clase btn-xx, donde xx es el tamaño, tambien si se desea que el boton abarque el 100% del el contenedor, se utiliza la clase btn-block.



Botones desabilitados

Los botones pueden tener un estado desabilitado, que no permite que sean accionados, y ádemas se ven de una manera distinta, para usarse se utiliza el atributo disabled.

Botones con Checkbox y Radio buttons

Los estilos de botones tambien pueden ser aplicados a distintas etiquetas que no necesariamente sean botones, como lo son los checkbox y los radio buttons, para hacerlo se necesitan las clases btn-group, btn-group-toggle, y el atributo data-toggle; dentro de este contenedor se abrega el checkbox, dentro de una etiqueta <label>, y en esta etiqueta se colocaran as clases btn y btn-xxxxx. 

<div class="btn-group btn-group-toggle" data-toggle="buttons">
