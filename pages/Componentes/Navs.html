Componentes Navs

La creación de barras de navegación se puede hacer de distintas manera, una de ellas es con listas, para ello es necesario agregar una lista con la etiqueta <ul> que tendra la clase nav y cada elemento de la lista <li> debera tener la clase nav-item y de contenido todos los hipervínculos con la clase nav-link. Tambien ya con la estructura de una barra de navegación, nosotros podemos agregarle estilos con CSS accediendo a las clases antes mencionadas.

Además de crear barras de navegación con listas, tambien se puede hacer con las etiquetas <nav>, agregandole la misma clase nav y dentro los hipervínculos con la clase nav-link.

Nota*: Se pueden crear navs verticales utilizando flexbox y con la clase flex-column en el contenedor con la clase nav.

Componentes Tabs

Las tabs funcionan de la misma forma que los navs, con la diferencia en los estilos que se le aplican a cada elemento, para crear las tabs se crear la estructura de las navs (tanto con listas como con la etiqueta nav) y, en el mismo contenedor se egrega la clase nav-tabs o nav-pills.

Se pueden utilizar algunas clases para brindarle mas estilos a nuestras barras de navegación, por ejemplo: nav-fill que ocasiona que los elementos ocupen el 100% de la columna.
