Componentes Listas

Para crear grupos de listas dentro del contenedor de los li se coloca la clase list-group y a cada list item la clase list-group-item. no es completamente necesario trabajar con listas, pueden usarse contenedores normales y de contenido hipervinculos y asi tener un grupo de lista de hipervinculos, si se trabaja de esta manera se puede colocar la clase list-group-item-action para darle un efecto de hover.

Para darle un poco mas de estilos se pueden usar las clases active para resaltar el elemento, disabled para desabilitarlo, y ádemas se pueden colocar colores a los list-item con la clase list-group-item-xxxxxx donde xxxxx es el color, veasé en Colores.

Para crear listas con un contenido deplegable, es necesario agregarle un ID a el contenedor list-group como tambien el atributo role="tablist", después a cada elemento de la lista o hibervinculos en este caso (aparte de las clases necesarias vistas anteriormente) es necesario agregarle los atributos, data-toggle="list", role="tab" y aria-controls="elemento#".

Posteriormente, en otra columna, se crea un contenedor con la clase tab-content y el id nav-tabContent y a cada elemento dentro de este contenedor se le agregaran las clases tab-pane, fade, show y active asi como el id que se utilizo como referencia en los elementos anteriore (ver el código).
