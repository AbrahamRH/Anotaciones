Componentes Alertas

Las alertas se crean con la clase alert, Se le pueden añadir estilos de boostrap con la clase alert-xxxx donde pueden ser, success,info, warning, danger, etc.

A las alertas se le pueden incluir cualquier otra etiqueta de HTML, para resaltar, o dar un enfasis en alguna parte de ella, como el subrayado o las negritas.
