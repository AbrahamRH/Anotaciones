Componentes Collapse

El componente de collapse siver, para mostrar contenido unicamente al activar un boton o un enlace de una carta u otro objeto.

Crear un componente collapse

Para crear un componente collapse, se necesita tener un enlace o boton referenciado a el id de el contenedor que tiene la información, este link tambien debe tener los atributos data-toggle="collapse", role="button", aria-expanded="false" y aria-controls="id".

Finalmenente al contenedor de la información que se mostrara se le coloca la clase collapse

Acordeón

Para crear un acordeón con el componente collapse,se necesitan añadir varias cartas, una por cada elemento que se deseé mostrar, veasé en cartas,las cartas necesitan estar dentro de un contenedor con un id. Las cartas necesitan tener un header y cada header su propio id y un boton para seleccionar el contenido desplegable de cada carta.

Este boton necesita tener los atributos de data-toggle="collapse", data-target="#idContenidoAdesplegar" aria-expanded="true" y aria-controls="#idContenidoAdesplegar".
