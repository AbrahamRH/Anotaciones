Componentes Carousel

Crear carousel

Para poder trabajar con un carousel en boostrap es necesario tener un contenedor con las clases carousel y slide asi como el atributo data-ride="carousel" y un id propio, en este contenedor es donde se añadiran las imagenes y botones.

Dentro del contenedor se debe crear uno nuevo con la clase carousel-inner donde se almacenaran las imágenes en contenedores con la clase carousel-item, para poder pasar entre imagenes una de ellas debe tener la clase active y ádemas antes del contenedor carousel-inner, se debe agregar una lista ordenada <ol> con la clase carousel-indicators donde cada item de la lista debe tener un atributo data-target="#id" y data-slide-to="#" donde # es el numero de la imagen qe se desea presentar.

Botones en el carousel

Despúes del contenedor carousel-inner es necesario colocar dos links apuntado a el id de el carousel con la clase carousel-control-xxxx y el atributo data-slide="xxxx" donde xxxx es prev o next según sea el caso. Dentro de cada link es necesario que se cree dos etiquetas <span>, dentro de la primera se deb agregar la clase carousel-control-xxxx-icon y el atributo aria-hidden="true", en el segundo <span> se debe ocupar la clase sr-only y de contenido (anterior/siguiente) según sea el caso.

Carousel con texto

Para agregar texto dentro de un carousel, dentro de cada carousel-item se debe agregar un contenedor con las clases carousel-caption y d-block, dentro de este contenedor se pueden agregar, titulos y párrafos para incluir texto.

Código completo

Para ahorrar un poco de tiempo, aqui esta el código de el ultimo carousel con todos los elementos.

						
<div class="carousel slide" data-ride="carousel" id="carousel3">
	<ol class="carousel-indicators">
		<li> data-target="#carousel3" class="active" data-slide-to="0"></li>
		<li> data-target="#carousel3" data-slide-to="1"></li>
		<li> data-target="#carousel3" data-slide-to="2"></li>
	</ol>

	<div class="carousel-inner">
		<div class="carousel-item active">
			<div class="carousel-caption d-block">
				<h4>>Titulo</h4>
				<p>Lorem ipsum dolor sit amet consectetur adipisicing.</p>
			</div>
			<img src="http://lorempixel.com/1800/500" alt="imagen" >
		</div>
		<div class="carousel-item">
			<div class="carousel-caption d-block">
				<h4>Titulo</h4>
				<p>Lorem ipsum dolor sit amet consectetur adipisicing.</p>
			</div>
			<img src="http://lorempixel.com/1801/500" alt="imagen" >
		</div>
		<div class="carousel-item">
			<div class="carousel-caption d-block">
				<h4>Titulo</h4>
				<p>Lorem ipsum dolor sit amet consectetur adipisicing.</p>
			</div>
			<img src="http://lorempixel.com/1802/500" alt="imagen" >
		</div>
	</div>
	<!-- Flecha izquierda -->
	<a href="#carousel3" class="carousel-control-prev" data-slide="prev">
		<span class="carousel-control-prev-icon" aria-hidden="true"></span>
		<span class="sr-only">Anterior</span>
	</a>
	<!-- Flecha derecha -->
	<a href="#carousel3" class="carousel-control-next" data-slide="next">
		<span class="carousel-control-next-icon" aria-hidden="true"></span>
		<span class="sr-only">Anterior</span>
	</a>
</div>
