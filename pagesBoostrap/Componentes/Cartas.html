Componentes Cartas

Creación de las cartas

Las cartas en Boostrap nos pueden ayudar para agredar información, imágenes y links, usando distintas clases para crearlas. Las cartas se crean mediante la clase card y dentro de ese contenedor se pueden usar las clases card-body, card-img-top y card-img-bottom.

Dentro de el cuerpo de la carta se pueden usar las clases cart-title, card-subtitle, card-text y card-link. Y dentro de la etiqueta card se puede cambiar el color del fondo con la clase bg-xxxx donde xxxx es el color, veasé en colores.

Nota*: Las imágenes deben colocarse fuera del cuerpo de la página.
