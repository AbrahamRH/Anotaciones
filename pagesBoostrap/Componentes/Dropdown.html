Componentes Dropdown

Para la creación de botones dropdown, se necesita un botón, veasé Botones, con un ID la clase dropdown-toggle y tambien necesita el atributo <data-toggle="dropdown">, despues de crear el botón, para agregar el contenido, es necesario crear un contenedor con la clase dropdown-menu con el atributo <aria-labelledby="ID">. Dentro de este contenedor puede ir un encabezado con la clase dropdown-header, y cada elemento debera llevar la clase dropdown-item.

Existen más clases para utilizar como la clase dropdown-divider que es utilizada para dividir los elementos que contiene el dropdown.

Encabezado

HTML CSS JS
Boostrap

Se pueden crear botones de dropdown con dos botones para cambiar la forma en que se ve, para esto hay que crear un grupo de botones, vease en Botones, el primero sera un boton normal con el titulo del dropdown, pero el segundo debera tener las clases dropdown-toggle y dropdown-toggle-split, su ID y el atributo <data-toggle="dropdown"> y de contenido de este segundo botón debera ir un span con la clase sr-only y un titulo.

Para agregarle el contenido, se hace igual que un dropdown normal.

Encabezado

HTML CSS JS
Boostrap
