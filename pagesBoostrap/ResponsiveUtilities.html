Utilidades para el diseño responsivo

Es necesario colocar la misma columna que se desea modificar varias veces, cambiando unicamente el atributo de tamaño en la columna para mostrarse en cada tipo de dispositivo.

Los atributos utilizados son:

  • d-none: Este atributo hace que no se muestre el elemento en pantalla, se pueden utilizar en distintos tamaños por ejemplo: d-sm-none.
  • d-block: Este atributo permite mostrar el elemento en pantalla, se pueden ocupar en distintos tamaños por ejemplo: d-sm-block.

Para usar estas utilerias es importante crear varias veces el elemento que se desea modificar de acuerdo a los tamaños para despues cambiar estos atributos en cada uno, deacuerdo a los tamaños que se deseen mostrar.

Nota*: Utiliza la herramienta para inspeccionar la página para ver los cambios en la columna de abajo de acuerdo a su tamaño, asi como sus atributos utilizados.
