Código

Código en linea

  • Se utiliza la etiqueta html de: code.

    • Lorem ipsum, dolor sit lorem amet consectetur adipisicing elit. Ipsam, tempora.

  • Para la creacion de etiquetas del tipo de HTML se ocupan simbolos de escape como & lt; para el símbolo < y & gt; para el símbolo >.

    • Lorem ipsum, dolor sit <lorem> amet consectetur adipisicing elit. Ipsam, tempora.

    Bloques de código

    Para escribir bloques de código se utilia la etiqueta <pre> posteriormente, dentro de esta misma etiqueta se utiliza la etiqueta <code> y dentro de esta última se colocan las lineas de código.

    Nota*: Todo dentro de la etiqueta <pre>, no debera estar indentado dentro de editor de texto ya que genera espacios en el contenido.

    Lorem ipsum dolor sit amet consectetur adipisicing elit. Sapiente, id?

    Indicar teclas a presionar

    Para indicarle al usuario unas teclas que se pueden utilizar su utiliza la etiqueta <kbd> mostrándose de la siguiente manera:

    Para el copiar texto presione: Ctrl + C

    Para el pegar texto presione: Ctrl + V
