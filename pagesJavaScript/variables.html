Variables

JavaScript tiene distitas formas de crear una variable, con distintas palabras reservadas, estas palabras reservadas son, var, let y const, las cuales vamos a describir a continuación.

Variables var

Las variables var, son utilizadas como variables globales, esto implica que pueden ser utilizadas en cualquier momento o bloque de código, incluyendo otros ficheros, por eso es mas recomendable utilizar las variables de tipo let, a menos que exista una razón muy específica, este tipo de variable puede ser inicializada en un valor opcionalmente.

Variables let

Las variables de tipo let, son variables con alcance de bloque (scope), que puede ser inicializada en un valor opcionalmente, al declarar una variable de tipo let, unicamente podra ser utilizada en el bloque en el que se declaro, ya sea una funcion, una estructura condicional o repetitiva, además de que no afecta a variables de otros ficheros con el mismo nombre.

Variables const

Con la palabra reservada const, crea una constante que debe ser inicializada siempre ya que esta es una constante y al igual que la variable let tiene un alcance por bloques.

Strings

Las variables de tipo String (o cadena de caracteres), se utilizan para guardar caracteres, palabras o frases de texto completas; las Strings en JavaScript pueden ser creadas de tres diferentes formas y todas ellas tienen las mismas caracteristicas de comportarse como arreglos en el posicionamiento de los caracteres:

let cadena1 = "Esta es una cadena de caracteres simple";
let cadena2 = String("Esta es una cadena de caracteres simple");
let cadena2 = new String("Esta es una cadena de caracteres simple");

Estas son las 3 maneras de declarar e inicializar una variable del tipo String, lo más importante a destacar aqui, es que la primera y segunda linea crean cadenas "primitivas" que unicamente guardan el texto, mientras que la tercera forma guarda la cadena como un objeto.

Metodos utilies para String

Las String tienen algunos métodos muy útiles, entre ellos estan:

indexOf() charAt() includes() concat()

indexOf()

Este método devuelve la posición de la letra que estes buscando o la posición de la primer letra de la parabra que se quiera encontrar, si no se encuentra la letra o la palabra retorna un -1.

Ejemplos

let cadena = String("Esta es una cadena de caracteres simple");
cadena.indexOf("es"); // 5
cadena.indexOf("E"); // 0
cadena.indexOf("felicidades"); // -1

charAt()

Este método devuelve el carácter en la posición que se indique.

Ejemplo

let cadena = String("Esta es una cadena de caracteres simple");
cadena.charAt(3); // a

includes()

Este método se encarga de devolver un valor booleano (verdadero o falso) si la palabra, letra ó frase se encuentran dentro de la cadena de caracteres.

Ejemplos

let cadena = String("Esta es una cadena de caracteres simple");
cadena.includes("es"); // true
cadena.includes("E"); // true
cadena.includees("felicidades"); // false
